# BaseWithMe
Base Application
##Getting Started
### Configure build.gradle (Project)
Add these lines:
```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
```
and:
```
dependencies {
  implementation 'com.github.User:Repo:Tag'
}
```
You can get version of this module at [Tag](https://jitpack.io/#ngtien137/BaseWithMe)
